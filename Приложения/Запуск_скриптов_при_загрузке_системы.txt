====== Запуск скриптов при загрузке системы ======

Сделать все свои скрипты и положить их в отдельный каталог.

Сделать файл:

'''
sudo mcedit /etc/rc.local
'''

Записать в этот файл построчно адреса всех скриптов, которые надо запускать:

{{{code: id="rc.local" lang="sh" linenumbers="True"
#!/bin/bash
/srv/………/myScripts/file1.sh
/srv/………/myScripts/file2.sh
exit 0
}}}

Чтобы не прерывать загрузку системы, надо обязательно указать успешное завершение скрипта без ошибок “''exit 0''” на случай, если какой-то скрипт исчезнет. 

Сделать файл исполняемым:

''sudo chmod +x /etc/rc.local''
