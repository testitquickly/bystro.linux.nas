====== Базовая настройка сервера ======

Из-под root выполнить 

''omv-firstaid''

Это делается сразу, пока к серверу ещё подключены монитор (по VGA) и клавиатура (USB).

===== Configure network interface =====

В норме подхватываются автоматически.

===== Configure web control panel =====

Поменять на „''81''”.

Это порт, по которому будет происходить обращение к серверу в сети.

Вероятно, позже захочется заходить на свой сервер снаружи по по IP вроде ''123.45.678.90:81'' Об этом ещё рано говорить, сперва надо настроить безопасность, но так или иначе, провайдер предложит выбрать любой порт, кроме 80, 25 и 255 — они по-умолчанию используются многими сервисами, поэтому сразу привыкаем к ''81''.

===== Пользователи =====

Здесь же из-под root можно (но не нужно) добавить на сервер новых пользователей. Если добавить их сейчас, то их «домашние страницы» сразу появятся на системном диске в разделе ''/home'', а нам надо позже сделать общий shared-каталог для всех профилей пользователей, поэтому отложим заведение пользователей для [[Users:Сделать “User home directory”]]
