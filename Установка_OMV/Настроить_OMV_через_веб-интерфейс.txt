====== Настроить OMV через веб-интерфейс ======

В браузере открыть ''http://192.168.1.20'' (IP сервера) и залогиниться как „admin” с уже хардкорным паролем.

Подавить желание переключиться с англоязычного интерфейса OMV на какой-либо иной.

	Есть и русский, и украинский, и китайский языки, но их переводы местами неадекватны и больше путают, нежели помогают. Особенно китайский.

	Кажется логичным сперва настроить несколько моментов, после чего один раз применить глобальное сохранение изменений, но лучше сохраняться после каждого шага. Каждый раз что-то может пойти не так.

===== General Settings =====

''System'' 
''> Workbench''

* Port = 81 (предустановлено),
* Auto logout = 60 минут минимум, а можно и ”весь день”.
* [Save]

Если NAS будет доступен из внешнего мира, надо настроить Secure connection — сперва надо где-то взять сертификат безопасности.

Если нет — сайт будет постоянно доступен по http, и этого достаточно.

===== Date & Time =====

'''
System 
> Date & Time
'''

* Часовой пояс = найти свой город
* Использование сервера NTP = да
* [Save / Apply / Yes]

===== Power Management =====

''System''
''> Power Management''
''> Settings''

* Monitoring = Yes
* Power Button = Standby (опционально)
* [Save / Apply / Yes]

===== Update Management =====

''System''
''> Update Management''
''> Settings''

 Install updates from = Community-maintained updates

===== Network =====

''Network''
''> General''

Тут значения, которые были заданы при установке базовой системы — ''tower.local''

''Network''
''> Interfaces''

Указано базовое сетевое устройство вроде ''enp2s0'' (Ethernet; может быть и Wi-Fi, если установлен).

Тыкнуть по нему мышкой ''> Edit''

''Network''
''> Interfaces''
''> Advanced settings''

==== MAC address ====

Указать действующий MAC address устройства.

	Узнать текущий mac address:

	''sudo ip a''

	Посмотреть зачение для ''link/ether'', например „''b4:99:ba:bf:48:44''“.
