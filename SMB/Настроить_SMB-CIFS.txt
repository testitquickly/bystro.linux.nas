====== Включить SMB/CIFS ======

''Services'' 
''> SMB/CIFS'' 
''> Settings''

General settings:
	* Enable = Yes
	* Workgroup = оставить «WORKGROUP»

Home directories:
	* Enabled = No
	* Set browseable = Yes

Advanced settings:
	* Use sendfile = Yes
	* Asynchronous I/O = Yes
