====== Добавить HDD ======

Обычно 
* в NAS вкладывают «чистые» диски, 
* затем встроенными средствами OMV на них делают таблицу разделов 
* и задают файловую систему, 
* делают каталоги, 
* делают эти каталоги shared 
* и заполняют данными.

В случе RAID-массива сценарий добавления/замены диска чуть сложнее, но суть та же: используются чистые диски, которые форматируются встроенными средствами.

===== Добавить «чистый» HDD =====

1. Вставить новый диск в NAS.
2. ''Storage > Disks'' — убедиться в том, что новый диск подключён.
3. ''Storage > File Systems > [+Create]'' — выбрать новый диск в выпадающем меню. Если это единственный новый диск, то в выпадающем меню только он и будет предложен.
	* File system = EXT4.
	* [Save / Apply / Yes]
4. Тыкнуть курсором по новому диску ''> [Mount]''

===== Добавить HDD с данными =====

1. Установить диски в NAS.
2. ''Storage > Disks'' — и убедиться в том, что они распознаваемы и готовы к труду и обороне.
3. ''Storage > Smart'' — и проверить их статус. Статус = зелёный, температура низкая.
4. ''Storage > File Systems'' — протыкать мышкой каждый диск последовательно > [Mount].

Убедиться в том, что в колонке «Mounted» диск отмечается как «Yes», а в колонке «Referenced» выбранный диск отмечен «No». Позже, когда на этом диске будет создан shared folder, значение «Referenced» сменится на «Yes».

===== Проверить состояние HDD =====

1. ''Storage > S.M.A.R.T. > Devices'' — проверить статус и температуру дисков,
2. ''Storage > File Systems'' — проверить свободное место на дисках.

===== Назначить права юзеру и группе =====

Переключиться под root:
'''

su 
'''

Получить список дисков:

''ls /srv/''

В этой команде сперва объявлена переменная ''MY_HDD'', в которой находится путь к корневому каталогу диска, на котором надо отдать всю власть юзеру ''astenix'' и членам группы ''users'' — чтение, запись и выполнение (''7''), а всем остальным — запретить всё (''0''), затем всё остальное:
'''

MY_HDD=/srv/dev-disk-by-uuid-3b52c281… && chown astenix:users -R $MY_HDD && chmod -R 770 $MY_HDD && echo -e "\nDone! — $MY_HDD"
'''
	
Здесь ''-R'' означает «рекурсивный», вглубь по дереву каталогов до полного финиша.

Подождать, пока команда отработает.

Объявить в переменной ''MY_HDD'' адрес следующего диска и повторить команду.
